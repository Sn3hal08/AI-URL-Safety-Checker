# AI-URL-Safety-Checker
Engineered a real-time AI-based URL security analysis platform capable of detecting phishing and malicious links using machine learning techniques. Built with Flask and a modern UI, the system provides instant risk assessment, enhancing user awareness and cybersecurity readiness.

## Features
- Real-time URL analysis
- Machine Learning model integration
- Modern responsive UI
- Flask backend

## Tech Stack
- Python
- Flask
- HTML, CSS
- Scikit-learn
